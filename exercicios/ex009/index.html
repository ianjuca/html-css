Trabalhando com Listas

Listas Ordenadas

O que tá marcado são as tags para utilizar nos códigos

Para listas ordenadas utilize a tag OL e
para listar os itens dentro dessa lista ordenada utilize LI
Podemos também escolher o tipo de numeração da lista,
se vai ser numérico (type ="1"), alfabético (type="A ou a") ou romano (type="I ou i")
Além disso podemos escolher de qual posição ela começa
basta usar (start="4") , nesse exemplo ela iria começar do número 4,
se quiser que comece de outro número é só trocar no lugar do 4.

Listas não Ordenadas

O que ta marcado são as tags para utilizar nos códigos

Para listas não ordenadas utilize a tag UL e
para listar os itens dentro dessa lista não ordenada utilize LI
Podemos escolher também o tipo de pontuação da lista,
para bolinha preenchida use (type="disc"),
para bolinha vazia use (type="circle") e
para quadrados use (type="square") . .
